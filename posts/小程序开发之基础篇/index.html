小程序开发之基础篇

发表于2021/07/12|更新于2021/07/12|Web

组件

视图容器

视图容器具有块级元素特点

view

原始视图容器参数比较少，因此比较轻量级。

scroll-view

可实现滚动效果的视图容器，需要固定高度或宽度，并且通过设置scroll-x或scroll-y参数指定滚动方向。

swiper

可实现滑动效果的视图容器，与swiper-item配合使用。可以用autoplay实现自动播放，并且可以通过interval设置切换间隔时间。

swiper-item

父级只能为swiper，它的宽高自动设置为100%。

基础内容

text

display: inline;

文本可以通过user-select决定内容是否可选，space决定空格是否连续。

icon

display: inline-block;

微信小程序有自带的图标，通过设置type的值来使用哪个图标，并且可以通过size与color设置样式。

表单组件

button

display: block;

微信小程序的按钮集成了一些方法来快速解决项目问题，例如可以通过form-type="submit"提交表单或者使用open-type="contact"打开客服会话，甚至可以通过disable禁用按钮。

input

display: block;

可以通过type设置输入框功能，还可以通过placeholder设置提示信息，通过maxlength设置输入框字符串最大长度。

form

display: inline;

可以配合具有form-type属性的元素使用。

导航

navigator

display: block;

页面链接可以规定页面的跳转方式，指定url通过open-type规定跳转规则。

媒体

image

display: block;

通过src获取图片对象并显示，通过mode规定图片在有限空间内不同的布局方式。当然它还提供一个非常好用的属性lazy-load，大大降低了页面渲染时间。
